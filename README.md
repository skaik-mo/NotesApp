# Notes App 📝

## Overview
Multilingual note-taking app supporting English and Arabic, with category organization.

## Features
- 🌐 Bilingual Support (English & Arabic)
- 📂 Note Categorization
- 💾 Local Data Storage

## Technology Stack
- **Architecture**: MVC Pattern
- **Language**: Swift
- **UI**: UIKit
- **Data Management**: Core Data
